Androton-Icons v0.1
==============

Icon set (>240 icons) combined from [Font Awesome](http://fortawesome.github.io/Font-Awesome/) and
[Android Stock icons](http://developer.android.com/design/downloads/index.html) in grey/white and
ldpi/mdpi/hdpi/xhdpi versions.

Made for [Androton](http://www.indiegogo.com/projects/androton-open-source/x/3133996) (development in progress).
